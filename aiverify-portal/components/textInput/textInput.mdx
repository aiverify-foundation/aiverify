<Title>TextInput React Component</Title>

You can use the `TextInput` component to create an plain text input field, number input field, or password input field.
Pass Tailwind CSS classes to the `labelClassName` and `inputClassName` props to override their styles. This is useful
for binding the input to Tailwind's dark mode.

Width of the component is deliberately set to be fluid (takes up the width of its container). So if you want to set a width for the component, put it in a container with a set width.

<Controls />

When writing tests for the `TextInput` component using React Testing Library, you can use the `screen.getByRole('textbox', { name: /your textbox/i });` 
selector, where "your textbox" is the value of the `label` prop.
